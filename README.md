# 2 変量正規分布のアニメーション

このリポジトリは、2 変量正規分布の同時確率密度関数を 3D でプロットし、相関係数が-1 から 1 まで変化するアニメーションを作成する Python コードを提供します。また、特定の \(X\) の値における \(Y\) の条件付き分布の期待値もプロットします。
